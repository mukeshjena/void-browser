I would like to introduce my android app to you

1. General Overview
•	App Name: Void Browser
•	Purpose: Void Browser is an ultra-lightweight, privacy-focused mobile browser designed to provide users with a fast, secure, and clean browsing experience. The app prioritizes user privacy by storing all data locally on the device, including bookmarks, history, and downloads. It features built-in ad-blocking, multi-tab management, and a discovery panel with news, recipes, weather, and images—all while maintaining a minimal footprint of just 16-19MB per architecture.
•	Target Audience: The app targets privacy-conscious individuals aged 18-50, including users with limited storage space on their devices, those seeking a lightweight alternative to heavy browsers, and anyone who values data privacy and wants a clean, distraction-free browsing experience without tracking or data collection.
________________________________________
2. Features & Functionality
•	Core Features:
o	Full web browsing with complete JavaScript support and WebView integration.
o	Built-in ad-blocking powered by EasyList filters to block intrusive ads and tracking scripts.
o	Multi-tab browser system with unlimited tabs and visual tab switcher (Chrome/Brave style).
o	Chrome-style search bar with secure/insecure connection indicators.
o	Discovery panel featuring news feed, recipes from TheMealDB, GPS-based weather forecasts, and beautiful images from Unsplash.
o	Reader mode for distraction-free reading by stripping away ads and clutter.
o	Bookmarks management and download manager with progress tracking.
o	Auto-hide AppBar that hides on scroll down and shows on scroll up for maximum viewing area.
o	Search engine selection (Google, Bing, or DuckDuckGo) for user preference.
•	Additional Features:
o	Automatic light/dark mode following system preferences.
o	Swipe gestures for back/forward navigation and pull-to-refresh.
o	Intelligent caching system for better performance and offline support.
o	State persistence across app restarts (tabs and sessions are saved).
o	Biometric fingerprint lock for app security.
o	QR code scanner for quick URL access.
o	Voice search support for hands-free browsing.
o	Download notifications with progress tracking.
o	Search history with smart suggestions.
•	User Flow:
1.	Users launch the app and are greeted with the discovery panel (news, recipes, weather, images).
2.	They can start browsing by entering a URL or search query in the Chrome-style search bar.
3.	Users can open multiple tabs, switch between them using the tab switcher, and manage their browsing sessions.
4.	They can save bookmarks, enable reader mode for articles, and download files with progress tracking.
5.	Users can access settings to configure ad-blocking, choose search engine, and customize privacy preferences.
6.	All data (bookmarks, history, downloads) is stored locally on the device for complete privacy.
________________________________________
3. Technical Details
•	Platform: Android (Android 5.0+ / API 21+), optimized for low-end devices with minimum 1GB RAM.
•	Tech Stack: Built using Flutter 3.10+ for cross-platform compatibility, Riverpod for state management, Hive and SharedPreferences for local storage, flutter_inappwebview for WebView functionality, and Dio for HTTP networking. The app follows Clean Architecture principles with Presentation, Domain, and Data layers.
•	Monetization:
o	Free model: The app is completely free with no ads, subscriptions, or in-app purchases.
o	All features are available without any limitations or premium tiers.
o	Focus is on providing value to users through privacy and performance rather than monetization.
